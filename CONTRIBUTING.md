# Contributing

## Setup

1. Clone the repository
1. Run `npm install` to install dependencies

## Running locally

```
npm run dev
```

## Run all checks

```
npm run check
```

## Tests

```
npm test
```

## Code style

```
npm run style
npm run style-fix
```

## Linting

```
npm run lint
npm run lint-fix
```

## Type checking

```
npm run types
```
